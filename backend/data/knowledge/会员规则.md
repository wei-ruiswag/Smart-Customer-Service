# 会员规则

## 1. 文档目的

本文件用于说明智选商城会员等级、成长值、会员权益、会员专享优惠、权益限制和异常处理规则，适用于会员咨询、订单优惠、售后优先级和客服回复场景。

## 2. 会员等级体系

### 2.1 等级划分

1. 普通会员：完成注册并通过基础身份校验的用户。
2. 银卡会员：满足一定消费金额、订单数量或活跃度要求的用户。
3. 金卡会员：具备较高消费频次、良好售后记录和稳定活跃度的用户。
4. VIP会员：达到平台高级会员标准，可享受更多专属权益和服务优先级。

### 2.2 等级计算方式

1. 会员等级可根据近一段时间内的有效消费金额、有效订单数量、账户活跃度和履约记录综合计算。
2. 取消订单、退款订单、异常交易、风险交易可能不计入成长值。
3. 平台可能在固定周期内更新会员等级，具体更新时间以会员中心展示为准。
4. 会员等级下降或权益调整时，用户可在会员中心查看原因说明。

## 3. 会员权益说明

### 3.1 购物权益

1. 会员可获得专属优惠券、品类券、生日券或活动券。
2. 部分会员可享受指定商品会员价，但会员价是否可与其他优惠叠加，以结算页为准。
3. 会员活动商品数量有限，可能出现售罄、限购或区域不可用情况。
4. 会员权益不得转让、售卖、出租或用于商业刷单。

### 3.2 售后权益

1. 高等级会员可享受售后咨询优先响应、工单优先排队等服务。
2. 售后优先权不等于售后必定通过，具体结果仍需根据商品状态和平台规则审核。
3. 部分会员可享受上门取件、免运费券或快速换新等权益，以页面展示为准。
4. 若商品存在高风险争议，仍需进入人工复核流程。

### 3.3 物流权益

1. 部分会员可获得运费券、包邮券或配送服务优惠。
2. 运费权益可能限制商品品类、配送区域、订单金额和活动时间。
3. 大件家电、偏远地区、特殊商品可能不适用普通运费权益。
4. 物流时效以实际配送能力和系统展示为准，会员身份不代表固定送达时间承诺。

## 4. 会员专享优惠规则

1. 会员专享券仅限符合会员等级的用户领取和使用。
2. 会员价商品以商品详情页和结算页展示为准。
3. 会员权益可能因活动、库存、地区、商品类型不同而差异化展示。
4. 用户会员身份过期后，未使用的部分会员权益可能失效。
5. 部分权益具有领取上限、使用上限、有效期和不可转让限制。

## 5. 会员权益限制

1. 会员不得通过刷单、虚假交易、恶意退款、批量注册账号等方式获取权益。
2. 会员不得将优惠券、会员价、运费券等权益转售给他人。
3. 若账号存在安全风险、异常售后或违规操作，平台可暂停部分会员权益。
4. 会员权益与普通售后政策发生冲突时，以平台规则和审核结果为准。

## 6. 会员售后与退款影响

1. 订单退款后，使用的会员优惠可能按系统分摊规则处理。
2. 若退款导致有效消费金额下降，可能影响后续会员等级评估。
3. 会员专享券是否退回，以券面规则、有效期和系统处理结果为准。
4. 因平台原因造成的权益异常，用户可提交工单申请人工复核。

## 7. 会员信息查询

1. 用户可在会员中心查看当前等级、成长值、权益、优惠券和有效期。
2. 智能客服可根据 user_id 查询用户等级，但不得展示完整手机号、地址等隐私信息。
3. 对会员权益有争议时，应提示用户提供订单号、活动页面截图或券编号。
4. 涉及账户安全、等级异常、权益冻结的场景，应转人工客服处理。

## 8. 客服回复规范

1. 客服应说明会员权益以会员中心、商品详情页和结算页展示为准。
2. 不得承诺会员必定享受某项优惠、必定售后通过或必定优先赔付。
3. 涉及会员等级和账户信息时，应核验 user_id 或会话身份。
4. 对异常权益、活动争议、系统显示不一致的问题，应创建工单并记录处理过程。
