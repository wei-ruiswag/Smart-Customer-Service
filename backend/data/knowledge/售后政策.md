# 售后政策

## 1. 文档目的

本文件用于规范智选商城在手机、耳机、笔记本、家电及电脑配件类商品中的售后处理标准，为智能客服系统、人工客服和用户提供统一的售后解释依据。

## 2. 适用范围

### 2.1 适用商品范围

1. 本政策适用于智选商城销售的手机、耳机、笔记本、家电、键盘、鼠标、鼠标垫、充电器、数据线、扩展坞等商品。
2. 商品页面、订单页面或活动页面另有特殊说明的，以商品页面展示的售后规则为准。
3. 定制类商品、激活后影响二次销售的数码商品、拆封后不适合退货的贴身使用类商品，可能不适用普通售后规则。

### 2.2 售后服务类型

1. 退货：用户将商品寄回平台或商家，平台审核通过后退还对应款项。
2. 换货：用户因质量问题、错发漏发、规格不符等原因申请更换同款或等价商品。
3. 维修：商品在保修期内出现非人为质量问题时，可申请维修服务。
4. 补发：商品出现配件缺失、赠品缺失、发票缺失等情况时，可申请补发。
5. 人工复核：系统无法自动判断责任或涉及高金额、高风险、争议较大的售后申请时，转人工客服处理。

## 3. 售后申请条件

### 3.1 基本条件

1. 用户应提供有效订单号，并确保申请售后的商品与订单信息一致。
2. 商品应保持完整，包括主商品、配件、说明书、保修卡、包装、赠品等。
3. 用户应如实描述问题，并根据客服要求上传商品照片、开箱视频、检测报告或物流凭证。
4. 售后申请应在平台规定的时限内发起，超过时限的申请将进入人工复核流程。

### 3.2 商品完好标准

1. 商品外观无明显划痕、破损、污渍、进液、烧毁、私自拆修痕迹。
2. 商品配件、包装、标签、序列号、保修凭证保持完整。
3. 手机、笔记本等设备应清除个人账号、锁屏密码、定位锁、云端锁等信息。
4. 耳机、鼠标、键盘等配件应保持配件完整，且不影响二次销售。

## 4. 不同品类的售后说明

### 4.1 手机类商品

1. 未激活、未使用且商品完好的手机，可按平台无理由退货规则申请退货。
2. 已激活手机如存在质量问题，应优先提供检测报告或官方售后检测凭证。
3. 因用户刷机、进液、摔落、私拆、改装导致的问题，不属于普通质量问题售后范围。

### 4.2 耳机类商品

1. 耳机属于近身使用商品，拆封后如无质量问题，可能不支持无理由退货。
2. 如存在单耳无声、连接异常、杂音、电池异常等问题，用户可申请检测、换货或维修。
3. 耳帽、充电仓、数据线等配件缺失时，应在签收后尽快联系平台处理。

### 4.3 笔记本类商品

1. 笔记本电脑如已联网激活、注册系统或绑定账户，可能影响无理由退货。
2. 如出现无法开机、屏幕异常、键盘失灵、硬件故障等问题，可申请质量问题售后。
3. 因自行拆机、加装硬件、刷写非官方系统导致的问题，可能影响保修。

### 4.4 家电类商品

1. 大件家电签收时应检查外包装、外观和配件，如发现破损应及时拍照并联系平台。
2. 需要安装的商品应由平台或品牌授权人员安装，自行安装导致的损坏可能影响售后。
3. 已安装使用的大件家电，如无质量问题，一般不支持普通无理由退货。

### 4.5 配件类商品

1. 键盘、鼠标、鼠标垫、充电器、数据线等配件如未使用且包装完整，可按规则申请退货。
2. 如出现按键失灵、连接异常、线缆破损、充电异常等质量问题，可申请换货或维修。
3. 易耗类配件、明显使用磨损配件，需结合商品状态进行人工复核。

## 5. 运费承担规则

1. 因商品质量问题、错发、漏发、运输破损导致的售后，平台或商家承担合理退换货运费。
2. 因用户个人原因发起无理由退货的，退回运费原则上由用户承担。
3. 用户使用到付、超标准快递或未经确认的特殊物流方式产生的额外费用，平台不承诺全部承担。
4. 大件商品上门取件、拆装、搬运费用按商品页面和售后审核结果确定。

## 6. 售后处理时效

1. 平台收到用户申请后，一般在1个工作日内完成初步审核。
2. 商品退回后，平台将在验收完成后给出退货、换货、维修或驳回结果。
3. 需品牌检测、物流核查、人工复核的售后申请，处理时间可能延长。
4. 智能客服不得承诺“必定通过”“一定赔付”“当天到账”等绝对结果，应以系统审核结果为准。

## 7. 风险与限制

1. 若用户存在恶意退货、调包、虚假凭证、重复滥用售后等行为，平台有权拒绝申请并转人工复核。
2. 涉及高价值商品、批量售后、异常地址、异常账号的申请，平台可要求补充证明材料。
3. 本政策为项目知识库示例，最终解释以平台页面展示和人工审核结果为准。
