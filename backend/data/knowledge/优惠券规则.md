# 优惠券规则

## 1. 文档目的

本文件用于说明智选商城优惠券的领取、使用、叠加、退回、失效和异常处理规则，适用于智能客服回答优惠券、满减、会员券、品类券和售后退款相关问题。

## 2. 优惠券类型

### 2.1 平台优惠券

1. 平台优惠券由智选商城统一发放，可在符合条件的商品或订单中使用。
2. 平台券可能限制品类、金额门槛、使用时间、用户身份和活动范围。
3. 平台券不得兑换现金，不得转让、售卖或用于非订单抵扣用途。

### 2.2 店铺优惠券

1. 店铺优惠券由店铺或商家发放，通常仅限对应店铺商品使用。
2. 店铺券的使用门槛、有效期和可用商品以券面说明为准。
3. 同一订单中是否可与平台券叠加，以系统结算页展示为准。

### 2.3 品类优惠券

1. 品类券适用于指定品类，例如手机、耳机、笔记本、家电、配件等。
2. 用户购买非指定品类商品时，系统不会展示该品类券。
3. 品类券可用范围可能随活动规则调整，以领取页面和结算页面为准。

### 2.4 会员专享券

1. 会员专享券仅限符合会员等级或活动资格的用户领取和使用。
2. 会员身份失效后，未使用的会员专享券可能无法继续使用。
3. 会员券不能保证长期发放，具体以活动页面展示为准。

## 3. 优惠券领取规则

1. 用户可通过活动页面、商品详情页、店铺页面、会员中心、客服活动入口领取优惠券。
2. 部分优惠券按先到先得方式发放，领完即止。
3. 部分优惠券需满足账号等级、会员身份、消费记录、活动区域等条件。
4. 每个用户在同一活动中可领取的数量可能有限制。
5. 优惠券领取成功后，可在“我的优惠券”或“卡券包”中查看。

## 4. 优惠券使用规则

### 4.1 使用条件

1. 订单商品、订单金额、活动时间和用户身份均满足条件时，优惠券才可使用。
2. 优惠券通常只能抵扣商品金额，是否可抵扣运费、服务费、安装费，以券面规则为准。
3. 优惠券不得用于提现、转账、购买虚拟违规商品或平台限制商品。
4. 优惠券应在有效期内使用，过期后自动失效。

### 4.2 使用顺序

1. 系统会根据订单可用优惠自动计算可用券。
2. 若存在多张可用券，用户可在结算页手动选择或按系统推荐选择。
3. 不同优惠之间是否叠加，以结算页最终展示金额为准。
4. 客服不得承诺某张券必定可叠加使用，应引导用户以结算页为准。

## 5. 叠加和限制规则

1. 同类优惠券一般不可无限叠加，具体以活动页面说明为准。
2. 平台券、店铺券、品类券、会员券是否可同时使用，由系统根据活动规则判断。
3. 预售商品、秒杀商品、补贴商品、特价商品可能不支持部分优惠券。
4. 若订单拆单，优惠金额可能按商品金额比例分摊到各子订单。
5. 若用户修改订单、取消部分商品或申请退款，优惠券分摊金额也可能发生变化。

## 6. 退款后的优惠券处理

1. 订单全额退款后，仍在有效期内且符合退回规则的优惠券可能返还账户。
2. 订单部分退款时，优惠券可能按系统规则分摊，已使用部分通常不直接退回。
3. 过期券、活动券、一次性补贴券、限时抢券，退款后可能不返还。
4. 如因平台原因导致优惠券无法使用，用户可提交人工复核。

## 7. 异常场景处理

1. 用户称有券但结算页不可用时，应检查商品范围、金额门槛、有效期、活动身份和地区限制。
2. 用户要求人工强制使用优惠券时，客服不得绕过系统规则承诺抵扣。
3. 用户通过异常方式获取优惠券、批量刷券、倒卖优惠券时，平台有权限制使用。
4. 涉及优惠券争议、活动页面截图和系统显示不一致时，应创建工单并转人工核查。

## 8. 客服回复规范

1. 客服应提示用户以优惠券页面、商品详情页和结算页展示为准。
2. 不得承诺“所有商品都能用”“一定能叠加”“退款后一定返券”等绝对表述。
3. 若用户提供订单号，客服可结合订单金额、商品类别和会员等级进行解释。
4. 涉及补偿券、人工券、异常券时，应进入工单流程，避免私下承诺。
