# 物流说明

## 1. 文档目的

本文件用于说明智选商城订单发货、配送、物流查询、地址修改、异常物流和签收注意事项，适用于订单查询 Agent、物流咨询场景和售后工单创建场景。

## 2. 订单物流状态说明

### 2.1 常见物流状态

1. 待发货：订单已支付或已确认，仓库尚未完成拣货、打包和交接。
2. 已发货：商品已交由快递公司或平台配送体系，用户可查看物流单号。
3. 运输中：包裹正在从发货仓、分拨中心或配送站点流转至收货地址。
4. 派送中：快递员正在安排最后一段配送，用户应保持电话畅通。
5. 已签收：包裹已由本人、代收点、门卫、驿站或授权人员签收。
6. 物流异常：可能出现地址不详、电话无法接通、天气影响、包裹破损、丢件、退回等情况。

### 2.2 查询方式

1. 用户可通过订单详情页查看物流公司、物流单号和最新轨迹。
2. 用户也可在智能客服中输入订单号查询订单状态和物流状态。
3. 如物流轨迹长时间未更新，用户可提交工单，由平台联系承运商核查。

## 3. 发货时效说明

### 3.1 普通商品发货

1. 现货商品通常在订单支付后按仓库处理顺序安排发货。
2. 活动大促、预售商品、定制商品、跨仓调货商品，发货时间可能晚于普通商品。
3. 商品页面标注预计发货时间的，以页面展示时间为重要参考。
4. 若订单中包含多个商品，可能出现拆单发货、分包裹配送的情况。

### 3.2 预售和缺货商品

1. 预售商品按尾款支付时间、商品页面承诺和商家备货进度安排发货。
2. 若商品缺货，平台可能提供等待、取消订单、换购或退款等处理方案。
3. 智能客服不得承诺未在页面展示的固定发货时间。

## 4. 配送时效说明

### 4.1 普通配送

1. 配送时效与收货地址、仓库位置、商品体积、天气、交通、节假日和承运商能力有关。
2. 同城或核心城市的部分现货商品可能较快送达，偏远地区、乡镇地区或特殊区域可能延长。
3. 页面展示的预计送达时间仅供参考，实际送达时间以物流轨迹和承运商配送为准。

### 4.2 大件家电配送

1. 冰箱、洗衣机、电视、空调等大件商品可能采用大件物流配送。
2. 大件商品可能需要提前预约配送时间，部分地区需确认是否具备上楼、安装或入户条件。
3. 因电梯、楼梯、门宽、道路限制导致无法配送入户的，用户需配合客服和物流协商处理。

## 5. 地址修改规则

1. 订单未发货前，用户可尝试申请修改收货地址，是否成功以系统审核结果为准。
2. 订单已发货后，地址修改难度较高，需由平台联系承运商尝试拦截或改派。
3. 已进入派送阶段的订单通常无法保证修改地址成功。
4. 为保障隐私安全，客服在展示地址时应进行脱敏，例如仅展示省市区和部分详细地址。

## 6. 签收注意事项

1. 用户签收前应检查外包装是否破损、变形、浸湿或有明显拆封痕迹。
2. 高价值商品、大件家电和易碎商品建议当面验收并保留照片或视频凭证。
3. 如发现少件、破损、错发，应尽快联系平台客服并提交相关凭证。
4. 已签收后发现问题，也可在售后时限内提交申请，但需要提供有效证据。

## 7. 物流异常处理

### 7.1 长时间未更新

1. 物流轨迹超过合理时间未更新时，用户可发起物流异常咨询。
2. 平台将根据订单号、物流单号和承运商反馈进行核查。
3. 若确认丢件或无法送达，平台可提供补发、退款或转人工处理方案。

### 7.2 派送失败

1. 派送失败可能由电话无法接通、地址不详、用户拒收、区域管制等原因造成。
2. 用户应及时核对收货电话和地址，并关注承运商通知。
3. 多次派送失败后，包裹可能退回发货仓或转入自提点。

### 7.3 包裹破损

1. 用户发现包裹外观破损时，应优先拍照留证并联系平台。
2. 如商品本身破损或无法正常使用，可提交售后申请。
3. 平台将结合照片、物流记录和商品验收结果判断责任。

## 8. 客服回复规范

1. 客服可以告知订单当前状态，但不得承诺超出物流系统显示的确定送达时间。
2. 涉及地址、手机号、姓名时，应进行必要脱敏。
3. 如物流问题涉及赔付、退款或补发，应以平台审核结果为准。
4. 对高价值商品、异常签收、疑似丢件等场景，应建议创建工单并转人工核查。
